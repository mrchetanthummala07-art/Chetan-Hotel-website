# 🏨 Chetan Hotel Website

## 📖 Project Overview
Chetan Hotel is a simple and responsive hotel website created using HTML, CSS, and JavaScript. The website provides information about the hotel, available rooms, a gallery, and an online booking form.

---

## 🚀 Features

- Home Page
- About Page
- Rooms Page
- Gallery Page
- Contact / Booking Page
- Responsive Design
- Navigation Menu
- Book Now Button
- Booking Form Validation using JavaScript
- Basic SEO (Title, Meta Description, Meta Keywords)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
